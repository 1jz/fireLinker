> A CLI tool that searches documents for the status of URLs/Links

## Features
Status codes supported
  <ul>
  <li>200</li> 
  <li>400 </li>
  <li>404</li>
  </ul>

URL Protocools supported
-HTTP
-HTTPS

Colors
red = bad URL
green = good URL
grey = unkown URL



## Install

```sh
npm i -g https://github.com/zg3d/fireLinker
```

## Usage

```sh
flink <FILE>
```
